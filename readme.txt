Free calculator to effortlessly evaluate the difference between dates. Easily add or subtract days from any date, with or without holidays and weekends.

[https://www.calculator.io/date-calculator/](https://www.calculator.io/date-calculator/)

== Usage ==

Add the Date Calculator shortcode to your page, post or sidebar:

`[ci_date_calculator]`

Add the following code to your template where you would like the Date Calculator to appear:

`<?php display_ci_date_calculator(); ?>`

== Installation ==

1. Upload the Date Calculator /ci_date_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Date Calculator plugin through the "Plugins" menu in WordPress.
